Mis proyectos

Encriptador

Encriptación básica: El encriptador convierte un texto ingresado en un formato cifrado usando una sustitución de caracteres. Por ejemplo, cambia vocales específicas por otras secuencias de caracteres predefinidas. Desencriptación: Además de encriptar el texto, la herramienta permite revertir el texto cifrado a su estado original, facilitando la comprensión del proceso de cifrado y descifrado.

Numero secreto

En el desafío del "Número Secreto", el programa genera un número aleatorio, y el usuario tiene que adivinar cuál es ese número mediante una serie de intentos. Cada vez que el usuario ingresa una respuesta, el programa le indica si el número es mayor o menor que el número secreto. El juego continúa hasta que el usuario adivina el número o agota el número de intentos permitidos.
